Below is a list of the permissions and what they do.

| Permission              | Description                                 |
|-------------------------|---------------------------------------------|
| ``mmoitems.admin``          | Permission for admin commands               |
| ``mmoitems.bypass.level``   | Bypasses the item level restriction         |
| ``mmoitems.bypass.item``    | Bypasses the item permission restriction    |
| ``mmoitems.bypass.ability`` | Bypasses the ability permission restriction |
| ``mmoitems.bypass.class``   | Bypasses the MMOCore class restriction      |
| ``mmoitems.update``         | Allows use of the /updateitem command       |
| ``mmoitems.edit.op``         | Allows admins edit op-item-stats listed in the config.yml |
