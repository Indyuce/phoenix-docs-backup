# 🔌 Other Plugins

MMOCore only has MythicLib as hard dependency, but all of our soft dependencies
are highly recommended for the RPG best experience.

## MMOItems 

This plugin is meant to go hand in hand with MMOCore. MMOItems provides item level & class restrictions, Item Stats, Quest Objectives, Triggers, Drop Table Items, and tons of other stuff. MMOItems is the only plugin which has built-in support, and it can't be beat! You do not need to install any compatibility addon, all the following features are automatically setup when the plugins detect each other. More info on [this wiki page](mmoitems.md).

## MythicMobs


MythicMobs adds new quest objectives and allows you to create an infinite amount of skills for MMOCore. Moreover, MMOCore adds new drop table items to MythicMobs. More info on [this wiki page](mythic.md).

## Fabled, ProSkillAPI, SkillAPI

Since MythicLib 1.3 you can also register custom skills in MythicLib, MMOCore and MMOItems using the SkillAPI skill creation system. More info on [this wiki page](../../mythiclib/skills/custom/fabled.md).

## Hologram Plugins

We support a lot of hologram plugins to allow for special holograms/indicators built into the plugin, for instance when fishing or gaining main class/profession experience. More info on [this wiki page](../../mythiclib/features/compatibility.md#hologram-plugins).

## WorldGuard, Residence

WorldGuard and Residence are recommended in order to utilize flag checks and skills. WorldGuard can be used to restrict fishing drop tables and block regen/restrictions to certain regions. You can find the full WG flag list [here](../../mythiclib/features/compatibility.md#flag-plugins).

## PlaceholderAPI


Used for all [placeholders](../general/placeholders.md).

## Vault

Vault is used when withdrawing physical currency from your balance or when depositing physical currency in your balance. More info on [this wiki page](../misc/economy.md).

## Citizens


Used in some [quest objectives](../features/quests.md) where players have to bring a specific
item back to a Citizen NPC.
