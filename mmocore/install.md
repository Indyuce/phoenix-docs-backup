# 🔌 Installation Guide

Many plugin issues are due to an invalid installations, so make sure you read the following tutorial to spare some time debugging!

1. Make sure you install [MythicLib](https://www.spigotmc.org/resources/mythiclib.90306/) first
2. Drag & drop the downloaded MMOCore file in your plugin folder.
3. Restart your server to generate default config files.
4. **UNINSTALL the MMOMana addon for MMOItems if you have it installed**, it is not meant to be used alongside any plugin that already has mana (like MMOCore).
5. MMOCore comes with default configs showcasing various compatibility features related to other plugins, including WorldGuard, Citizens, MythicMobs, MMOItems... If these plugins are not installed, you will get some startup warnings in your console, which is completely normal. Take your time to explore these features and install the required plugins if you want to use them.
