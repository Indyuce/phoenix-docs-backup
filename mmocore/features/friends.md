# 👯‍♀️ Friends

The friends system built into MMOCore is a simple yet effective system. When using the `/friends` command, it will pull up a GUI that shows all your current friends and options to add more. 

The stats shown when hovering over an online player include class, level, and online time. For an offline player, you will see how long they have been off!

![PEVbnk9](uploads/friend_ui.gif)

![6bKKTKJ](uploads/friend_online.png)

You can disable the friend module by simply removing the `/friends` command in the `commands.yml` config file
