# 🧑‍⚖️ Permissions

| Permission                    | Description                      |Default   |
|-------------------------------|----------------------------------|----------|
| ``mmocore.admin``                 | Grants access to ``/mmocore``            |OP        |
| ``mmocore.waypoints``             | Access to ``/waypoints``             |OP        |
| ``mmocore.pvpmode``               | Access to ``/pvpmode``               |OP        |
| ``mmocore.currency``              | Access to ``/deposit`` and ``/withdraw`` |OP        |
| ``mmocore.bypass-waypoint-wait``  | Bypass waypoint waiting time     |None        |
| ``mmocore.class-select``          | Access to ``/class``                 |OP        |
| ``mmocore.attributes``            | Access to ``/attributes``            |Player          |
| ``mmocore.skills``                | Access to ``/skills``                |Player          |
| ``mmocore.profile``               | Access to ``/profile``               |Player          |
| ``mmocore.friends``               | Access to ``/friends``               |Player          |
| ``mmocore.guild``                 | Access to ``/guild``                 |Player          |
| ``mmocore.party``                 | Access to ``/party``                 |Player          |
| ``mmocore.skilltrees``            | Access to ``/skilltrees``            |Player          |
| ``mmocore.quests``                | Access to ``/quests``                |Player          |
