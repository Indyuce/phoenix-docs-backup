# 🏠 Home

![Vyor7Op](uploads/banner.png)

From the creator of MMOItems, we bring you a whole new RPG solution! Replacing outdated and popular plugins, we aim to provide users the same customization and features their old favorite RPG plugins used to offer. After over 7 years of development and extensive bug testing, we hope to provide the best experience possible!

::: tip
Get MMOCore from [Polymart](https://polymart.org/product/3412/mmocore) or [Spigot](https://www.spigotmc.org/resources/mmocore.70575/)
:::


If something is not quite clear, please join our [Discord](https://phoenixdevt.fr/discord) server to get help from staff or community! We do our best to keep this wiki up to date. Let us know if you see a typo or anything outdated!

::: info
Note that MMOCore comes with a basic set of existing classes, attributes, quests... There are many examples provided to help you get started, but you will need to configure the plugin mostly yourself.
:::

Navigate to the next page when you are ready to install MMOCore!
