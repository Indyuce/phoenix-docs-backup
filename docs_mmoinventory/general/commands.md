# 🖥️ Commands

| Command | Description | Permission |
|---------|-------|------------|
| `/mmoinventory` | Opens the custom inventory if enabled | `mmoinventory.open` |
| `/mmoinv reload` | Reloads the plugin |  `mmoinventory.admin` |
| `/mmoinv export-data` | Exports player data from flat storage to SQL |  `moinventory.admin` |
| `/mmoinv <player>` | Open a player's inventory (you can interact with their items) | `mmoinventory.open.other` |

The following aliases work for the main command: `/mmoinv`, `/rpginv`, `/rpginventory`
