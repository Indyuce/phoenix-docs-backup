# 🤔 What is MythicLib?

You might be wondering - **I installed MMOItems or MMOCore, I already have enough plugins on my server, why do I need to install and worry about an additional plugin?**

MythicLib is a little more than just a library that contains shared code and utilities. It's a plugin that centralizes features to reduce config duplication and code redundancy.

It centralizes fundamental RPG features such as:
- the MMO player stats system,
- the MMO damage and element system,
- damage and regen indicators,
- the MMO skill system and scripting engine,
- integrations with other plugins,
- many smaller-size shared config options,
