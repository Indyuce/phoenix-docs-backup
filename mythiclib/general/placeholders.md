# 🏷️ Placeholders

These are placeholders that you can use in both MMOItems and MMOCore.

| **Placeholder**                                 | **Description**                                               |
|-------------------------------------------------|----------------------------------------------------------|
| `%mythiclib_stat_<stat_name>%`         | Current value of a player stat.                |
| `%mythiclib_raw_stat_<stat_name>%`         | Current value of a player stat with no rounding.                |
| `%mythiclib_cooldown_<cooldown_key>%` | This placeholder handles cooldowns for MANY things including skills from MMOCore/MMOItems, MI items, mitigation mechanics etc. See the list of available cooldown keys [here](https://gitlab.com/phoenix-dvpmt/mythiclib/-/wikis/Variables#cooldownmap). |
| `%mythiclib_defense_damage_reduction%` | Get damage mitigated by defense when inflicted 100 damage. |
| `%mythiclib_space_<number>%` | Positive (`10`)/negative (`-10`) space using [Negative Space Font Resource Pack](https://www.spigotmc.org/threads/negative-space-font-resource-pack.440952/). |
