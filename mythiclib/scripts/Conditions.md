Scripts perform specific actions which are called **mechanics**. A script is a list of mechanics and conditions. A script only executes when all its conditions are met! They are checked in order, which means MythicLib will first evaluate the first condition and stop there if it's no met. The script then executes all its mechanics in order, one after the other.

| Condition Type | Description |
|---------------|-------------|
| [Generic](https://gitlab.com/phoenix-dvpmt/mythiclib/-/wikis/Conditions:-Generic)     | Math formulas, propositional expressions... |
| [Location](https://gitlab.com/phoenix-dvpmt/mythiclib/-/wikis/Conditions:-Location) | Location based conditions |
| [Miscellaneous](https://gitlab.com/phoenix-dvpmt/mythiclib/-/wikis/Conditions:-Miscellaneous) | Other |
