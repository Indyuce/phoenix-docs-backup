A script is a list of mechanics and conditions. If all of the script conditions are met, then its mechanics are ran in order, from first to last. A mechanic is a specific action, it can be sending a projectile, sending a message to a player, playing a sound to multiple entities, etc.

| Mechanic Type | Description |
|---------------|-------------|
| [Buffs](https://gitlab.com/phoenix-dvpmt/mythiclib/-/wikis/Mechanics:-Buffs)     | Give temporary stats; restore health, food... |
| [Miscellaneous](https://gitlab.com/phoenix-dvpmt/mythiclib/-/wikis/Mechanics:-Miscellaneous) | Commands, cooldowns, delay, entity effects... |
| [Movement](https://gitlab.com/phoenix-dvpmt/mythiclib/-/wikis/Mechanics:-Movement) | Give velocity and teleport |
| [Offense](https://gitlab.com/phoenix-dvpmt/mythiclib/-/wikis/Mechanics:-Offense) | Deal damage, apply potion effects... |
| [Raytraces](https://gitlab.com/phoenix-dvpmt/mythiclib/-/wikis/Mechanics:-Raytraces)     | Cast invisible rays that stop on blocks and/or entities |
| [Shapes](https://gitlab.com/phoenix-dvpmt/mythiclib/-/wikis/Mechanics:-Shapes)        | Helixes, slashes, parabolas... |
| [Variables](https://gitlab.com/phoenix-dvpmt/mythiclib/-/wikis/Mechanics:-Variables)     | Edit/create numbers, vectors etc. |
| [Visual](https://gitlab.com/phoenix-dvpmt/mythiclib/-/wikis/Mechanics:-Visual) |  Messages, sounds and particles |
| [External](https://gitlab.com/phoenix-dvpmt/mythiclib/-/wikis/Mechanics:-Other-Plugins) |  Mechanics implemented by other plugins |
