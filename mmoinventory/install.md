# 🔌 Installation Guide

- Download and install [MythicLib](https://www.spigotmc.org/resources/mmolib-mythiclib.90306/). It is the only hard dependency for MMOInventory.
- Download and install the MMOInventory JAR file in the `/plugins` folder.
- We recommend reading [this wiki page](compatibility/mmoitems.md) if you are running MMOItems on your server.
- Download the default MMOInventory server resource pack and install it (or merge it with your existing server pack).
