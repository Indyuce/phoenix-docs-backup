# 🧙‍♂️ Supported RPG Plugins



All these RPG plugins are supported by the `class{..}` or `level{..}` slot restrictions:
- AureliumSKills (level only)
- BattleLevels (level only)
- Heroes
- mcMMO (level only)
- McRPG (level only)
- MMOCore
- RacesAndClasses
- RPGPlayerLeveling (level only)
- SkillAPI and ProSkillAPI
- Skills and SkillsPro

You can edit what plugin MMOInventory should hook onto in the main plugin config file
```yml title="config.yml"
level-module: mcmmo
class-module: skillapi
```
