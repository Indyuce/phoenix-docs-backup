# 🏷️ Placeholders

| Format | Usage | 
|--------|----------------|------------|
| `%bountyhunters_top_name_<rank>%` | Rank must be between 1 and 16. Returns the Xth player on the hunter leaderboard |
| `%bountyhunters_top_bounties_<rank>%` | Claimed bounties of some player in the leaderboard |
| `%bountyhunters_top_heads_<rank>%` | Amount of successful bounties of some player in the leaderboard |
| `%bountyhunters_top_level_<rank>%` | Player level from the leaderboard |
| `%bountyhunters_level%` | Displays the personal hunter level |
| `%bountyhunters_successful_bounties%` | The amount of bounties the player successfully claimed |
| `%bountyhunters_illegal_streak%` | The amount of illegal kills the player got since last respawn |
| `%bountyhunters_illegal_kills%` | Total amount of players killed illegally (ie no bounty on target) |
| `%bountyhunters_current_bounty%` | Current bounty on the player's head ||
| `%bountyhunters_title%` | The current hunter title (or 'No Title' if none) |
| `%bountyhunters_progress%` | The hunter progress bar (amount of bounties he must claim before leveling up) |
| `%bountyhunters_before_level_up%` | Amount of bounties the player must claim before leveling up |
