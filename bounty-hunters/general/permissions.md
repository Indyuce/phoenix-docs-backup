## Player Permissions

| Format | Description |
|--------|-------------|
| `bountyhunters.list` | Access to `/bounties` |
| `bountyhunters.add` | Permission to create or increase a bounty. |
| `bountyhunters.claim` | Permission to **claim a bounty**. |
| `bountyhunters.remove` | Lets you remove your own bounties. |
| `bountyhunters.leaderboard` | Access to the bounty hunter leaderboard GUI. |
| `bountyhunters.title` | Access to hunter titles. |
| `bountyhunters.animation` | Access to bounty animations. |
| `bountyhunters.redeem-heads` | Access to `/redeembountyheads` |

## Admin Permissions

| Format | Description |
|--------|----------------|
| `bountyhunters.admin` | Access to admin commands. |
| `bountyhunters.update-notify` | Be notified when joining the server when a new update is available. |
| `bountyhunters.immunity` | No player shall place a bounty on you. |
| `bountyhunters.immunity.bypass` | Bypass previous bounty immunity. |
| `bountyhunters.untargetable` | No player shall target you. |
| `bountyhunters.untargetable.bypass` | Bypass previous target immunity. |
