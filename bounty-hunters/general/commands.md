# 💾 Commands

## Player Commands

| Usage | Description |
|----|----------------|
| ``/addbounty <player> <amount>`` | Creates a bounty/adds money to current bounty. |
| ``/bounties`` | Displays current bounties. |
| ``/hunterlb`` | Displays the bounty hunter leaderboard. |
| ``/bounties titles`` | Displays owned titles. |
| ``/bounties animations`` | Displays owned bounty animations. |
| ``/redeembountyheads`` | Redeem heads from your victims. |

## Admin Commands

| Usage | Description |
|----|----------------|
| ``/bounties help`` | Displays the help page. |
| ``/bounties remove <player>`` | Removes a bounty on a player. |
| ``/bounties edit <player>`` | Edit a bounty (edit reward & contributors). |
| ``/bounties reload`` | Reloads the plugin config files. |

When performing ``/addbounty`` as an admin, you may add ``-bmm`` at the end of your syntax to entirely bypass the minimum/maximum bounty reward limit. You may also use ``-nt`` to entirely negate the bounty creation tax.
