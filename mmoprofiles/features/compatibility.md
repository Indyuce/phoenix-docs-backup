# 🦾 Compatibility

The main feature of this plugin is to enable other plugins to use its API and also have profile-based data. For now, this is only the case for MMOCore, MMOInventory and CoreTools but we hope that other plugins will make the step to be integrated with MMOProfiles. A plugin that supports MMOProfiles will have its data saved for the profile and not for the player, and changing profile will be as if you logged in with a different account. 

## Vault Economy
MMOProfiles does save the player's Vault balance. This way any Vault-based economy plugin will be supported by MMOProfiles.

## Phoenix Plugins
[MMOCore](https://gitlab.com/phoenix-dvpmt/mmocore/-/wikis/home), [MMOInventory](https://gitlab.com/phoenix-dvpmt/mmoinventory/-/wikis/Home), [MMOItems](https://gitlab.com/phoenix-dvpmt/mmoitems/-/wikis/home) and [CoreTools](https://gitlab.com/Tanerx/CoreTools/-/wikis/home) **FULLY SUPPORT** profile-based data storage. We plan to enable some features to persist when changing profiles, such as MMOCore friends, but this is not currently the case.
