# 💾 Commands

| Command | Description |
|---------|-------------|
| `/mmoprofiles reload` | Reloads the plugin and config files. Watch out, some options require a server RESTART. |
| `/mmoprofiles open <player-name>` | Opens profile selection for a player. This should only be used when the `no-gui-on-login` option is toggled on. |
| `/mmoprofiles export-data` | Exports player data from flat storage to SQL |
